T1 - Introdução ao HTML

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas web. É a espinha dorsal de todo site na internet.

O que é HTML?

HTML é uma linguagem de marcação que usa "tags" para estruturar o conteúdo de uma página. Cada tag tem uma função específica.

Estrutura Básica

<!DOCTYPE html>
<html>
<head>
    <title>Título da Página</title>
</head>
<body>
    <h1>Meu Primeiro Título</h1>
    <p>Meu primeiro parágrafo.</p>
</body>
</html>

Principais Tags

  • <h1> a <h6> - Títulos de diferentes níveis
  • <p> - Parágrafos
  • <a> - Links
  • <img> - Imagens
  • <div> - Divisões de conteúdo
  • <span> - Elementos em linha
